niveau des lib utiles, a part lodash :

  • moment.js manipulation de dates quand l’objet natif suffit pas, un peu lourd dans le client, mais OK dans node (pour un truc plus light voir instadate)
  • mz Permet d’avoir des promise plutot que des callback pour les librairie native de node, for pratique
  • request-promise Une api un peu moins barbare pour faire les requetes HTTP, pratique quand tu a besoin de gerer des login, connexion Oauth etc pour des crawler un peu complexe, ca devient vite relou a la main la gestion de cookie / session persistence inter requetes. Request gere tout tres bien, c’est la version promisifié comme on dit. (node uniquement, le navigateur a fetch)
  • AVA Pour faire du TDD
  • webpack Pour avoir require et tout bundle ton JS dans le navigateur
  • postcss Pour survivre le css permet de rajouter tout un tas de goodies au css. j’utilise principalement : variables, color, calc, nesting, minify et l’indispensable auto-prefixer pour pas avoir a ecrire les -webkit -moz etc etc…
